Questions Fréquentes

Comment utiliser SilentiumX ?

SilentiumX est un Tool Open-Source en Python. Installez Python depuis python.org, en cochant "ADD PYTHON TO PATH", puis redémarrez votre PC. Ouvrez ensuite le "Setup" pour utiliser le Tool.

Les mises à jour sont-elles avertie ?

Oui, toutes les mises à jour sont averti d'une notification automatique. Nous nous engageons à maintenir et améliorer constamment l'outil pour vous offrir la meilleure expérience possible.

Le logiciel est-il sécurisé ?

Absolument ! Le tool est sécurisé il ne contient aucun malware. Les antivirus peuvent bloquer les fichiers mais il ne sagit de rien de dangereux.
